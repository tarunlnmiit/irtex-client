Introduction

This Image Retrieval system helps you to understand the WHY? behind the images retrieved for a query image by providing supported textual and visual explanations

How to use it? 

→ Simple search

  • Select the dataset from the dropdown. We have selected two most familiar datasets which are CIFAR-10 and PASCAL VOC. Click on Begin to start searching.
  • From the left panel, select any image as a query image and system will start looking for similar images.

→ Downstream Search

  • If you want to use any retrieved image as a query image, then go ahead and select the image of your choice. The system will automatically retrieve results for this new query image

→ All about Explanations

  1. Click on the Explain button provided against each retrieved image. The dialog box will provide you textual as well as visual explanations for each retrieved result against every feature employed. You can also switch between each feature tab to get better understanding.
  2. The area where the query image selected is shown, a global textual explanation is generated to overall explain the reasons for the retrieved results.
  3. To compare retrieved images with each other, select the checkbox and click on ‘Click to compare’. The dialog box visualises feature comparison for each selected image.
  4. Next to each retrieved image, we provide an explanation in terms of horizontal bars which tells you the similarity value with the query for every feature on hover over the names of the features.

→ Explore the system 

  1. To have a look at more images please navigate between pages by clicking on the page navigation bar provided on upper right corner above retrieved images
  2. You can also increase/decrease the number of retrieved images shown on the current page by changing the value in ‘Items per page’ next to the page navigation bar.
  3. To get images on the basis of individual feature only, please select the desired feature from the ‘Rank by’ dropdown
